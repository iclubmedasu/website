# iClub Platform
 
A members portal for university clubs with project management, certificate issuance, and team coordination features.
 
## Applications
 
| App | Description | URL |
|-----|-------------|-----|
| Members Portal | Private portal for club members | localhost:3001 |
| Public Website | Public facing club website | localhost:3002 |
| Backend API | Express REST API | localhost:3000 |
